# Présentation

Ce mod a été fait à partir de PsychEngine, moteur de Friday Night Funkin. Dans ce mod, vous incarnez Didi2 et vous devez battre vos différents rivaux dans des battles de raps.
Chaque week fait référence à une des séries de la chaîne Youtube Didi2 FR. Affrontez Didi666, Baldi, Ralph, Mr Freez et Mr Hacker qui seront vos ennemis principaux.

***
